Truss No-Print No-Screen

DEMO

1 - This is a paragraph. It displays on the screen as normal font- weight and also prints bold.

2 - This is also a bold paragraph. It does not print, only displays on the screen as no bold.

3 - This small red text with yellow background prints but does not display on the screen. (Some printers don't print background colors.)
4 - This image and this text displays on screen but does not print.

5 - Everything in this div displays on the screen but does not print (unless class overrides the div).

6 - This sentence displays on the screen and also prints.
7 - This sentence of the same paragraph, however, does not print because it has class no-print.
8 - This sentence does not display on the screen, but does print because it has class no-screen.
9 - This last sentence of the paragraph both displays on the screen and prints. But only the print is bold.
